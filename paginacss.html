CSS






Para empezar vamos a empezar sabiendo que son las siglas de CSS, al español son hojas de estilo en cascada, junto con HTML sirve para darle diseño a las páginas, nos ayud a mejor la vista del usuario para que así esta sea más atractiva para su vista CSS nos ofrece muchas herramientas de diseño y edición las cuales nos ayudan en muchos aspectos al momento de juntarlo con HTML. Los estilos en línea son declaraciones CSS que se integran en las etiquetas HTML mediante el atributo style. Este método tan solo afecta al elemento en el que se integra el código. El CSS en línea es complicado de entender y mantener ya que mezcla los estilos CSS con el código HTML.





Forma de usar CSS


como tal no hay una forma exacta de usar CSS, pero si hay unas reglas que se deben seguir, como lo son las menciones al momento de hacer las ediciones o arreglos, por ejemplo al usar una clase se tiene que tener en cuenta que siempre se le llamara a la clase por medio de un punto, pero en cambio, cuando se usa un id la forma de llamarla es distinta a la clase, este se termina llamando con un numeral y despues el nombre que se le asigno



Datos de edicion


CSS nos ofrece la opción de poder cambiar la tipografia de una página al igual que el color y su tamaño, todo esto es gracias al usar de forma
directa en el documento HTML la palabra style, esto nos ayudara a poder hacer lo que se menciono anteriormente, un ejemplo es un "font-family, que"
que sirve para la tipografía o un "font-size" que ayuda en el tamaño de la letra, se podría cambiar el color de fondo con un "background-color".

Los mencionados anteriormente son unos ejemplos básicos de la gran cantidad
de cosas que se pueden hacer con CSS.






Sintaxis

La sintaxis de CSS es el conjunto de reglas que definen cómo escribir código en CSS. Estas reglas definen cómo se deben estructurar las reglas de estilo y cómo se deben combinar los selectores, propiedades y valores para producir un diseño coherente en una página web. La sintaxis de CSS es bastante sencilla, y generalmente se escribe en archivos separados para evitar mezclar el código HTML con el CSS.
