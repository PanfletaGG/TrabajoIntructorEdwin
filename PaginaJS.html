JAVASCRIPT




JavaScript es un lenguaje de programación de secuencias de comandos que se utiliza para hacer que las páginas web sean interactivas y dinámicas. Fue inventado por Brendan Eich en 1995 y se ha convertido en uno de los lenguajes de programación más populares del mundo. JavaScript se utiliza para crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás en una página web. avaScript es un lenguaje de programación orientado a objetos y multiplataforma que se puede aplicar a un documento HTML y usarse para crear interactividad dinámica en los sitios web. Con JavaScript, puedes implementar funciones complejas en páginas web y mostrar oportunas actualizaciones de contenido, mapas interactivos, animación de gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo y mucho más






Semantica


La estructura básica de JavaScript se compone de elementos comunes de código como enunciados condicionales, bucles, funciones y eventos. Una función JavaScript es un bloque de código que realiza una tarea específica y se puede reutilizar. Una función JavaScript tiene un nombre, argumentos y sentencias. El nombre identifica la función, los argumentos son los datos que recibe la función y las sentencias son las instrucciones que ejecuta la función. En JavaScript, también hay una biblioteca estándar de objetos como Array, Date y Math, y un conjunto básico de elementos del lenguaje como operadores, estructuras de control y declaraciones 2. Los tipos de datos integrados disponibles en JavaScript incluyen números, cadenas, booleanos, objetos y matrices

Java para desarrollo web


El uso más popular de JavaScript es para el desarrollo web, y es un a de las herramientas más poderosas que un desarrollador puede tener en su manos. Los desarrolladores usan JavaScript en esta área para añadir interactividad y funciones que mejoren la experiencia del usuario y hagan a internet mucho más disfrutable. JavaScript se ha expandido más allá del desarrollo de interfaz, que es donde comenzó. Recientemente, JavaScript ha llegado al back-end, o dorsal de desarrollo web. Esto quiere decir que los desarrolladores tienen acceso de interfaz a métodos CRUD (Create, Read, Update, Destroy; en español: Crear, Leer, Actualizar, Destruir) y hasta puede utilizarse en el motor de un sitio web.
