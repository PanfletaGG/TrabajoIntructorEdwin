HTML






Para empezar tenemos que saber que es HTML, sus siglas en español es "lenguaje de marcado de hipertexto", html nos ayuda a hacer la estructura o esqueleto de una página, ayudandonos a divivdirla en distintas secciones como lo es la cabeza o "head", el cuerpo o "body" y el "footer" que es la parte de abajo que se ve en la página. Aquí podremos ver distintos datos básicos sobre que es HTML.






Estructura


Entre las partes básicas que podemos ver en una página podemos Encontrar la cabeza, que es la parte superior que se ve en una página, la barra de navegación que en si son los enlaces que nos van a dirigir a otras partes de la página, por lo general está hecha con botones, despues estaría el contenido principal de la página que por lo general mantiene la gran parte de contenido de la página, y por ultimo estaría el pie de página o footer.

Etiquetas en html:

Title
Se utiliza para definir un título para el documento.

Body
Define la parte corporal o cuerpo de un documento.

P
Define un trozo de texto como párrafo.

B
Define un texto concreto en negrita.

Sub
Define el texto subíndice.

Select
Permite definir una lista desplegable.

Table
t Permite la creación o definición de una tabla junto con otras etiquetas como tr, tt y td.

Img
Permite insertar o definir una imagen personalizada, al usar la etiqueta de style de css podemos mejorar el diseño.

a
Permite añadir un vínculo mediante la ayuda de un href, src y alt, que sirve para que ambas se complementen.

Span
Define una sección del documento.



Styles en HTML y como usarlo de forma interna, externa y en linea

Se utiliza un CSS en línea para aplicar un estilo único a un solo elemento HTML. Un CSS en línea usa el style atributo de un elemento HTML. El style de forma interna se utiliza para definir un estilo para una sola página HTML. Un CSS interno se define en la head sección de una página HTML, dentro de un style elemento. Y el style externo se utiliza mediante una hoja de estilo externa para definir el estilo de muchas páginas HTML. Para usar una hoja de estilo externa, agregue un enlace en la head sección de cada página HTML.
